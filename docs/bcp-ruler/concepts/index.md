# Concepts

* [Complexity Element](complexity-element.md) — One atomic, countable thing found in a story
* [Element Router](element-router.md) — Assigns every element to exactly one dimension
* [Identity Rule](identity-rule.md) — Per-dimension definition of what makes one element distinct
* [Sizer](sizer.md) — Per-dimension step that assigns a Size/Score to routed elements
* [Dimension Module](dimension-module.md) — Code shape: prompt + response shape + decision-table scoring
* [NFR Gate](nfr-gate.md) — Standard expectations score 0 in NFR dimensions
* [Maturity Score](maturity-score.md) — Confidence indicator accompanying every BCP total
